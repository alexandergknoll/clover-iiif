# Thumbnail

The Thumbnail component is used to display a thumbnail image for a work.

<IIIFBadge href="https://iiif.io/api/presentation/3.0/#thumbnail" />

## React Usage

```jsx
import { Primitives as IIIF } from "@samvera/clover-iiif";

const CustomThumbnail = ({ thumbnail }) => {
  return <IIIF.Thumbnail label={thumbnail} />;
};

export default CustomThumbnail;
```
